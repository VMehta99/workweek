#workweek
A darktheme weekly calendar for your work week, built with React.

### Instructions
1. Visit https://workweek.herokuapp.com/
2. Click on the Day
3. Add Title, Time(optional), Event Description(optional), Priority(optional) 
    - Remember to include commas!
